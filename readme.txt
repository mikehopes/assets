运行环境：Thinkphp3.1.2框架，建议运行环境win7 32bit，Mssql2008，wampserver2.5-Apache-2.4.9-Mysql-5.6.17-php5.5.12-32b；
功能说明：运行需要修改实际数据库配置（home/conf下的config.php），代码文件放在wamp安装目录的www文件夹下；
admin登陆密码111111，登陆进入admin管理台；其他用户登录进去固定资产管理主界面；首次登录必须修改密码；
支持对用户不同模块操作权限的管理（新增资产、修改资产、删除资产、转移资产）；
支持用户对不同公司的权限管理；
修改资产使用人必须通过转移操作实现；
可以查看历史转移记录；
支持用户和资产编号模糊查询；

2、IT固定资产统计；
